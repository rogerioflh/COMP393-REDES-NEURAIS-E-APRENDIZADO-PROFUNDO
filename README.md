# CNN Classification Assignment

This workspace contains an implementation of the attached assignment:

- `cnn_classification_assignment.ipynb`: Google Colab notebook with all three required scenarios.
- `reports/final_report_template.md`: optional report structure if a separate PDF report is needed.
- `scripts/generate_cnn_assignment_notebook.py`: reproducible generator for the notebook and report template.

## How to use

1. Open `cnn_classification_assignment.ipynb` in Google Colab.
2. Select a GPU runtime.
3. Run all cells from top to bottom.
4. After the runs finish, complete the markdown discussion/reflection answers using the generated metric tables, confusion matrices, examples, and Grad-CAM figures.

The notebook uses TensorFlow/Keras, CIFAR-10, DermaMNIST, and Oxford-IIIT Pet. The epoch values are collected in the configuration cell so the experiments can be shortened for testing or extended for stronger final results.
